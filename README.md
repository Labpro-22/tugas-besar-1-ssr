# Tugas Besar 1 IF2010 Pemrograman Berorientasi Objek

### How to build

Dependensi:
1. CMake build system
2. Raylib

Instalasi Raylib dapat dilakukan dengan mengikuti panduan pada README repositori resmi: https://github.com/raysan5/raylib

Program nimonspoli dapat di-build dengan menjalankan command berikut,
```bash
cmake -S . -B build
cmake --build build
```

Setelah proses build selesai, akan muncul executable pada directory bin/ yang dapat langsung dijalankan,
```bash
bin/nimonspoli
```
